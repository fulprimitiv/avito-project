# Тестовое задания по направлению Frontend для Avito

## Технологиии
- Проект написан на **TS** с структурированной архитектурой — типы в `src/shared/types`.
- Собрка - **Vite**. До этого в проектах использовал Webpack, конфиг приходилось настраивать, лоадеры добавлять + Babel использовать, в Vite c этим попроще.
- **Prettier + ESlint** для единого форматирования кода
- **React + react-router-dom** 
- **Axios** - работа с бэком (`src/api/axios.ts`, `src/api/adsService.ts`).
- Верстка при помощи **SCSS** с методологией **БЭМ** для большинства компонентов.
- Адаптивный дизайн (декстоп / таблет / мобилка).
- **MUI + Recharts** использовал для фильтров / галереи / диаграмм.
- **Файловая организация:** хуки, типы и константы в `src/shared`, компоненты разбиты на `list-components` / `item-components` / `global-components`  / `stats-components`.

## Навигация

- **Хедер:** путь `/stats` при клике на ФИО модератора; на `/list` - логотип Avito.
- Страница списка объявлений (`/list`) с карточками, фильтрами (статус, категория, ценовой диапазон, поиск), сортировкой и пагинацией (10/страница).
- Детальная страница (`/item/:id`) с галереей, полным описанием, таблицей характеристик, информацией о продавце и историей модерации. Навигация: "Назад к списку", "Предыдущее"/"Следующее".
- Страница статистики (`/stats`) - карточки метрик и графики активности / распределения решений / по категориям.

## Установка и запуск приложения

### Установка зависимостей и запуск (Backend)
```bash
cd tech-int3-server
npm install
npm start
```
Сервер будет доступен по адресу: http://localhost:3001

### Установка зависимостей и запуск Frontend
```bash
cd tech-int3-app
npm install
npm start
```
Приложение будет доступно по адресу: http://localhost:5173
