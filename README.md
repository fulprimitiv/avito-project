# Тестовое задания по направлению Frontend для Avito

## Установка и запуск Backend части приложения

### Установка зависимостей
```bash
cd tech-int3-server
npm install
```

### Запуск в режиме разработки
```bash
npm start
```
Сервер будет доступен по адресу: http://localhost:5173


## Установка и запуск Frontend части приложения

### Установка зависимостей
```bash
cd tech-int3-app
npm install
```

### Запуск в режиме разработки
```bash
npm start
```
Приложение будет доступно по адресу: http://localhost:3001
